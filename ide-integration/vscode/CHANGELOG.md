# Changelog

All notable changes to the "Changelog Hub" extension will be documented in this file.

## [1.0.0] - 2024-12-26

### Added

- Initial release
- Compare API specifications (OpenAPI, AsyncAPI, GraphQL, gRPC)
- Git integration for comparing with any ref
- Analyze API specifications
- Validate API specifications
- Generate changelogs from Git history
- Inline diagnostics for deprecated endpoints and issues
- Hover information for endpoints and schemas
- Quick fixes for common issues
- API Explorer view
- Breaking Changes view
- Changelog view
- Status bar integration
- Configurable settings
- Context menu integration
